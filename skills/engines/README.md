# Engine Skills

Skills for creating, extracting, and maintaining Rails engines.

## Skills

- **[create-engine](create-engine/)** — Scaffold and configure new Rails engines.
- **[test-engine](test-engine/)** — Setup dummy apps and engine-specific spec suites.
- **[review-engine](review-engine/)** — Audit engine quality and coupling.
- **[release-engine](release-engine/)** — Versioned releases, changelogs, and upgrade notes.
- **[document-engine](document-engine/)** — Writing comprehensive engine documentation.
- **[create-engine-installer](create-engine-installer/)** — Idempotent install generators for host apps.
- **[extract-engine](extract-engine/)** — Incrementally extracting code from host app to engine.
- **[upgrade-engine](upgrade-engine/)** — Maintaining cross-version Rails and Ruby compatibility.
